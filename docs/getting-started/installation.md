# Installation Guide

## Prerequisites
- Ensure you have [Node.js](https://nodejs.org/) version 14 or later installed.
- Git must be installed on your machine.
- A code editor (like VSCode) recommended for modifying files.

## Installation Instructions
1. **Clone the Repository**  
   Run the following command in your terminal:  
   ```bash
   git clone https://github.com/rafaelsistems/aksara.git
   ```

2. **Navigate to the Project Directory**  
   Change into the directory by executing:  
   ```bash
   cd aksara
   ```

3. **Install Dependencies**  
   Use npm to install the required packages:  
   ```bash
   npm install
   ```

## Verification Steps
1. **Start the Application**  
   After installation is complete, you can start the application with:  
   ```bash
   npm start
   ```

2. **Access the Application**  
   Open your web browser and navigate to `http://localhost:3000` to verify the application is running.

## Troubleshooting
- **If `npm install` fails:**  
   - Make sure your Node.js version is correct. You can check it by running `node -v` in your terminal.  
   - Ensure you have an active internet connection.

- **If the server does not start:**  
   - Ensure that you are running the command from the root directory of the project.  
   - Check for any error messages in the terminal to diagnose issues.
